# Chroma Drop Spec

## Core fantasy and 10-second hook
Players drop colored blocks into a grid, aiming to create matches of 4+ same-colored blocks for a satisfying clear. Quick matches generate immediate score bursts within 10 seconds.

## Q&A
- **Core fantasy:** Color-matching drop puzzle with smooth physics and satisfying clear effects.
- **Hook:** Quick, satisfying clears under 10 seconds.
- **Loop:** Daily seeded piece sequences, combo multipliers, leaderboard.
- **Session length:** 30s‑2m bursts.
- **Skill vs Luck:** Skillful placement determines combos; deterministic seed ensures fairness.
- **Fail-state fairness:** Grace piece buffer, clear feedback, no abrupt punishments.
- **Difficulty ramp:** Piece speed increases, new colors unlocked, obstacles introduced.
- **Distinctive mechanic:** Zen‑like physics with smooth block settling and visual bloom.
- **Art scope:** Minimalist palette, simple shapes, low asset count.
- **Audio:** Soft chimes on clears, ambient loop.
- **Monetization:** Non‑intrusive ads, optional skin purchases.
- **Technical constraints:** Target Android 5+, <30 MB APK, 60 FPS.
